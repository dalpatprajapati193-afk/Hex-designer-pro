# 🛡️ Shell & Tube HEX Designer Pro
High-fidelity thermal design & rating tool powered by **NIST (CoolProp)**. 

### Key Features:
* **Phase Change & Sensible Heat** support.
* **Automated Property Tracking** (Cp, Latent Heat, P, T).
* **LMTD & Physical Sizing** (TEMA logic).
* **Process Data Sheet** generation.
